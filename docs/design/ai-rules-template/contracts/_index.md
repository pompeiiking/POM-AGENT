# 接口契约索引

> **本目录由人类（架构师/技术负责人）维护，AI不得擅自新增或修改契约文件。**
> 适用方案：方案B（分布式，模块数≥10）

## 使用说明

1. 每对需要对接的模块创建一个 `[模块A]-[模块B].contract.md` 文件
2. 全局共享类型放在 `shared-types.contract.md`
3. 每次新增契约后，在下方索引表中登记
4. AI开发时根据索引找到相关契约并严格遵守
